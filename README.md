# 🔢 Number Guessing Game (Python)

A simple and addictive **Number Guessing Game** where the player tries to guess the correct number chosen by the computer.  
The player can choose a custom difficulty by selecting the maximum range.

---

## 🚀 Features

- User chooses the upper limit (e.g., guess between 1–50, 1–100, 1–500, etc.)  
- Computer randomly picks a number in this range  
- Program gives hints:
  - “Greater than”
  - “Smaller than”
- Tracks number of attempts  
- Play Again option  
- Beginner-friendly Python logic  

---

## 🕹️ How to Play

1. Run the script  
2. Enter the maximum number for the guessing range  
3. Keep guessing until you find the correct number  
4. Number of attempts will be shown  
5. Choose **Y/N** to play again  

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/NumberGuessingGame.git
cd NumberGuessingGame
```

Run the game:

```bash
python guess.py
```
